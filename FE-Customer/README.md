# FE-Customer - Giao diện người dùng cửa hàng cá cảnh

Giao diện dành cho khách hàng của hệ thống quản lý cửa hàng cá cảnh.

## Tính năng

- 🏠 Trang chủ với sản phẩm nổi bật
- 🐟 Xem danh sách và chi tiết sản phẩm
- 🛒 Giỏ hàng
- 💳 Thanh toán
- 📦 Lịch sử đơn hàng
- 👤 Thông tin cá nhân

## Cài đặt

```bash
npm install
```

## Chạy ứng dụng

```bash
npm run dev
```

Ứng dụng sẽ chạy tại `http://localhost:3001`

## Build

```bash
npm run build
```
